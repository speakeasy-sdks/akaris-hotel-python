# ImageSizeEnum

Indicates the size of the image. Hospitality APIs no longer support thumbnail


## Values

| Name          | Value         |
| ------------- | ------------- |
| `LARGE`       | Large         |
| `MEDIUM`      | Medium        |
| `SMALL`       | Small         |
| `THUMBNAIL`   | Thumbnail     |
| `EXTRA_LARGE` | ExtraLarge    |
